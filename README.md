👗 MOCHI Fashion Store App

Welcome to MOCHI, a modern mobile fashion e-commerce application built with Flutter and powered by Firebase.
This project delivers a smooth and stylish shopping experience where users can explore trendy fashion products, manage their cart, place orders, and track purchase history in real time.

Designed with a clean UI and seamless user experience, MOCHI demonstrates the integration of modern mobile development technologies including:

✨ Flutter UI Development
🔥 Firebase Authentication
🛒 Shopping Cart & Checkout System
📦 Real-Time Firestore Database
📱 Responsive Mobile Interface

This project was developed as part of a Mobile Application Development coursework to showcase practical implementation of frontend and backend mobile technologies.
